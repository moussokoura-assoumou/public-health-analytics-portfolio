## About This Portfolio

This portfolio showcases my data analytics skills applied to real-world public health scenarios, built with interactive Power BI-style dashboards using Chart.js and HTML/CSS.

It demonstrates my ability to design, build, and communicate data insights across four key public health domains.

---

## 📂 Dashboards

| Dashboard | Description |
|---|---|
| 🦠 Disease Surveillance | Case incidence trends, district distribution, outbreak investigation log |
| 🏥 Infection Control | HAI rate trends, MDRO pathogens, IPC compliance by facility |
| 📈 Program Performance | COVID-19 ops volume, data quality scoring, Prior Auth analytics |
| 🔬 Epi Analytics | Epidemic curves, age-stratified attack rates, odds ratio forest plots |
